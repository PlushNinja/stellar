    class Section

    Sections are page subdivisions defining the layout of its content segments. The segments contain the actual content and have optional associations to external resources like images, products, promotions, etc.

    The layout for a section is selected from presets that specify how many contained segments are created.

def duplicate
  dup.tap do |c|
    c.page = nil
    columns.each do |column|
      c.columns << column.duplicate
    end
    pictures.each do |picture|
      c.pictures << picture.duplicate
    end
  end
end
