    module ContentGateway

    ContentGateway::OikotieAsunnot implements a content gateway interface to the Oikotie Asunnot API. Documentation is at <api.asunnot.oikotie.fi/>.

    To use the gateway, initialize a ContentGateway::OikotieAsunnot object with a store argument that contains the oikotie_asunnot_api_key and oikotie_asunnot_broker_id attributes needed for accessing the API.

    Calling the feed method will access the API to load a collection of cards as ContentItem objects. The feed method accepts options to control the desired card type and number of items.
