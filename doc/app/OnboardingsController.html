    class OnboardingsController

    This controller takes a visitor through the onboarding process modelled in the Account::Onboarding module. At each step, the current action selects the onboarding subclass, which specifies the validations and other details. The forms will post to the validate action that terminates the process by calling create when there is no next step.

    Checking for registration is done when the onboarding module requires it, sending the visitor to the registration process if not logged in. Returning to where they left off is done by storing the location for the next step as set by the last time validate was called.

def create
  store = @onboarding.finalize!(current_user)
  session[:onboarding_attributes] = nil
  session[:next_step] = nil
  redirect_to controller: 'store', action: 'index', host: store.primary_host.fqdn
end

def validate
  current_step = params[:step]
  @onboarding = onboarding_for_step(current_step)
  @onboarding.attributes = onboarding_params
  session[:onboarding_attributes] = @onboarding.attributes

  if @onboarding.valid?
    next_step = onboarding_next_step(current_step)
    create and return unless next_step
    session[:next_step] = url_for(action: next_step)
    redirect_to session[:next_step]
  else
    render current_step
  end
end
