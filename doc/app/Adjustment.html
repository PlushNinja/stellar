    class Adjustment

    Adjustments are modifiers affecting the price of an adjustable object. Anything adjustable may have an adjustment that adds amount_cents to its price. The attached label provides detailed information. If an adjustment has an associated source, it should respond to `description` that defines the contents of the label.

def amount_sans_tax_cents
  return nil if amount_cents.nil?
  if account_for_taxes? && price_includes_tax?
    amount_cents - tax_cents
  else
    amount_cents
  end
end

def amount_with_tax_cents
  return nil if amount_cents.nil?
  if account_for_taxes? && !price_includes_tax?
    amount_cents + tax_cents
  else
    amount_cents
  end
end

def tax_cents
  return nil if amount_cents.nil?
  if price_includes_tax?
    amount_cents * tax_rate / (tax_rate + 100)
  else
    amount_cents * tax_rate / 100
  end
end
